# Code & Lab Wiki

A working notebook of bioinformatics analyses, pipelines, scripts, and lab procedures — for myself and anyone else who finds it useful.

## Sections

- [Bioinformatics](bioinformatics/index.md) — pipelines, analyses, one-off scripts.

## How this site works

Pages are Markdown files under `docs/`. Push to `main` and GitHub Actions rebuilds the site automatically. See the repo's `README.md` for local preview and contribution notes.
